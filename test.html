测试任务 User Testing

任务一：定制外貌，身份绑定

  • 选择您最想要的宠物种类，比如鳄鱼或者猩猩，当然您可以选择狗或者猫。
  • 就像游戏捏脸系统一样，请对您的宠物外貌进行设计与调整，也可提供一张画像或者三维模型，系统会自动生成相似的宠物外貌。
  • 您的宠物将会对您的人脸、气味和动作等特征进行采集，从此宠物将与您本人进行绑定。
  • 如果您想克隆自己已故的宠物，可以提供宠物生前的视频信息，新的机器宠物可以通过视频来学习您的宠物的生活习性和动作特征

    • 任务二：互动，语音助手

  • 和宠物互动，除了散步和喂食，您还可以与宠物对话。
  • 宠物可以完全理解您的语言，您可以轻松命令它去取快递和外卖，您可以发出任何您能想到的命令。

    • 任务三：云端共享宠物数据

  • 您可以将宠物的数据上传云端进行保存，可以更换宠物的实体而保持灵魂不变。
  • 当您出差到一个新的城市，可以去共享宠物中心拷贝您的宠物的数据，让您的私人宠物永远陪伴在您身旁。

    评估报告 Evaluation Report

    用户们对宠物的外貌设计功能都很满意，但是对于机器宠物的内核的设计还涉及到很多问题。我们之后会对宠物的核心功能设计进行改进，可以让用户关闭自己不需要的冗余功能模块。除此之外，我们对宠物的数据部分进行改进，防止因为太过智能而导致智械危机，我们将把智能更多的运用到宠物本身的习性学习等模块。最后，我们将对产品的安全和隐私模块进行改进，防止在使用过程中泄漏个人隐私，也防止一些别有用心的人利用机械宠物危害他人安全。
